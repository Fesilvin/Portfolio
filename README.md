# Personal Portfolio Website

Персональный сайт-портфолио, разработанный с использованием HTML, CSS и JavaScript.

## О проекте

Сайт представляет собой персональное портфолио с информацией:
- обо мне;
- навыках;
- проектах;
- контактных данных.

Проект выполнен в минималистичном стиле с адаптивной вёрсткой и анимацией появления секций при прокрутке страницы.

## Используемые технологии

- HTML5
- CSS3
- JavaScript
